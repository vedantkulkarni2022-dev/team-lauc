Linear Algebra Theory

Linear Dependence

A set of vectors {v₁, v₂, ..., vₙ} is linearly dependent if there exist scalars c₁, c₂, ..., cₙ, not all zero, such that:

c₁v₁ + c₂v₂ + ... + cₙvₙ = 0

If the only solution is c₁ = c₂ = ... = cₙ = 0, the vectors are linearly independent.

Rank Test

The rank of a matrix formed by the vectors determines dependence:

  • If rank < number of vectors → Linearly Dependent
  • If rank = number of vectors → Linearly Independent

For n vectors in ℝᵐ: if n > m, vectors must be dependent.
